# DownloadInstallFromURL
A quick and dirty script to download latest versions of software from a web URL
This exists so that we do not have to package common software constantly, and stay up to date.

Tips on using this.....

You can use this script in 3 ways..

Hardcode varibles into the fields indicated and run the script locally or with ARD
You can hardcode the values and make it a post install script in a packge file to be installed any way that you want. ( Jamf users think patch managment )
You can run it from JAMF with dynamic variables passed as parameters to the script
